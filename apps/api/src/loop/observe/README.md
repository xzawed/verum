Stage 6 of The Verum Loop — collecting OpenTelemetry-compatible traces and spans from all `verum.chat()` and `verum.retrieve()` calls in connected services.

OBSERVE is the only stage that requires the connected service to be running. All other stages (especially ANALYZE) operate without service execution.
